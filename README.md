Tekken: Emilie De Rochefort, Responsive Coded Interface Project
A responsive webpage used to showcase a biogrpahy based on a video game characters characterstics, abilities, fighting style, background, etc.
Specfically:
Character biography (age, background, lore)
Fighting Abilities (How the character fights, possibly coercing a user to play as the character or to steer away from them)
Interactive Links (links used to gain new and other information for further understanding)
CSS Design (color pallette made to match the characters external aesthetics)
